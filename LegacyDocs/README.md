# Legacy J2O documentation
Here you can find the legacy documentation for J2O (formerly known as JIPipeRunner) prior to the JIPipe 6.0.0 integration update. You can find the up-to-date documentation on [the J2O documentation website](https://example.com/). 

> [!WARNING] 
> If you are a legacy user switching to the new version, be sure to remove the old names from the omero.web.apps and omero.web.ui.right_plugins config and run the installation again!
